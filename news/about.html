关于全球新闻日报

本项目每日自动聚合来自全球50+家主流媒体的新闻头条，涵盖政治、经济、科技、环境、体育等多个领域。

由 Python 脚本抓取 RSS feed，通过 GitHub Actions 每日自动运行，构建静态网站并托管于 GitHub Pages。

技术栈

  • Python: feedparser, aiohttp, BeautifulSoup4
  • GitHub Actions 每日定时运行 (UTC 00:00)
  • GitHub Pages 静态托管
  • 纯前端 HTML + CSS + JS，零外部依赖

数据来源

聚合自以下50家全球主流媒体：

  • Reuters, AP, BBC, CNN, Al Jazeera, France 24, DW, Euronews
  • NYT, Washington Post, WSJ, Bloomberg, NPR, LA Times, USA Today, NBC, ABC, CBS, Guardian, Economist, Time, Forbes
  • Guardian UK, Telegraph, Independent, FT, Le Monde, El Pais, Spiegel, Irish Times, The Times
  • SCMP, Straits Times, Japan Times, Nikkei, The Hindu, Times of India, Korea Herald, Asahi
  • Xinhua, CCTV, 环球网, 澎湃, 观察者, 新浪, 腾讯, 参考消息
  • Sydney Morning Herald

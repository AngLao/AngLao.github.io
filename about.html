关于全球新闻日报

本项目每日自动聚合来自全球50+家主流媒体的新闻头条，涵盖政治、经济、科技、环境、体育、文化等多个领域。

由 Python 脚本通过 RSS feed 抓取新闻，经 GitHub Actions 每日自动运行，构建静态网站并托管于 GitHub Pages。

技术栈

  • Python: feedparser, aiohttp, BeautifulSoup4
  • GitHub Actions - UTC 00:00 每日自动运行
  • GitHub Pages 静态托管
  • 纯前端 HTML + CSS + JS，零外部依赖

数据来源（50+家媒体）

覆盖全球主要通讯社、北美、欧洲、亚太、中国、大洋洲的主流新闻媒体，包括：Reuters, AP, BBC, CNN, Al Jazeera, France 24, DW, Euronews, NYT, Washington Post, WSJ, Bloomberg, NPR, Guardian, Economist, FT, Le Monde, El Pais, Spiegel, SCMP, Nikkei, Xinhua, CCTV, 环球网, 澎湃, 观察者, 新浪, 腾讯, 参考消息 等。
